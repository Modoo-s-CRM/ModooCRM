# ModooCRM

⚙️ModooCRM : 내담자관리프로그램입니다. 
- 상담 일정
- 내담자 관리
- 내담자 조회
- 증상·월별 내담자·상담 유형 등 통계
- 월별 상담사 내담자 관리

⚙️ 담당 파트
* 프론트엔드 : 이용만
* 백엔드
  이용만) 메인페이지, 특이사항, 회원관리, 회원 상세 조회, 상담 이미지 자료 관리, 상담 일지 관리, 통계
  김정근) 월별 상담사 내담자 관리
